# Breast Cancer Detection – Machine Learning & Deep Learning Project

A multi-modal diagnostic system designed to classify breast cancer cases using both traditional machine learning and deep learning techniques.

## Technologies Used

- Python
- Scikit-learn
- TensorFlow
- Keras
- NumPy
- Pandas

## Models Implemented

### Machine Learning
- Logistic Regression
- Decision Trees
- Random Forest (with balanced class weights)

### Deep Learning
- Artificial Neural Network (ANN)
  - Optimized dense layers
  - Dropout regularization
  - Cross-entropy loss

## Key Achievements

- Achieved strong classification accuracy using Random Forest
- ANN model outperformed traditional ML approaches on test data
- Implemented full ML pipeline including preprocessing and evaluation
